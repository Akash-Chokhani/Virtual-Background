# Virtual Background

## Overview

The Virtual Background is a software application designed to replace or enhance the background behind the person in a video stream. Virtual Background allows you to blur or change your background to a preset or custom images or videos of your choice.


## Table of Contents

* [Features](#features)
* [Installation](#installation)
* [Usage](#usage)


## Features

* **Blur Background**: Hide your background by blurring it
* **Background Replacement**: Easily replace your background with a virtual one.
* **Custom Background**: Use your own images or videos as backgrounds.
* **Real-Time Effects**: Apply background changes in real-time


## Installation

To install and use Virtual Background, follow these steps:

1. **Clone the repository**:
```
git clone https://github.com/Akash-Chokhani/Virtual-Background.git
cd Virtual-Background
```

2. **Install dependencies**:
```
pip install -r requirements.txt
```

3. **Run the application**:

Execute the Virtual Background application:
```
cd src
python main.py [OPTIONS]
```


## Usage

1. **Customize your background**:

You can customize Virtual Background by passing valid OPTIONS argument during execution of application.

Run the following code to get the usage information:
```
python main.py --help
```
or
```
python main.py -h
```

2. **Enjoy Your Virtual Background**:

Your background is now replaced with the chosen virtual background, providing a professional and engaging appearance.
